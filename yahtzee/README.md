# APL Yahtzee

Conservative dialect compatible with most APL implementations, including APL\360, IBM 5100, VS APL, APL/PC,
APL2, SHARP APL, APL*PLUS, iAPL, microAPL, Dyalog, etc.

Source file character encoding is UTF-8.

Saved (binary) workspace files for several APL systems will be included.
